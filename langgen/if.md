## Podmínka `if`

Podmíněné vykonání kódu při splnění podmínky.

### python
```python
if x > 0:
    print(x)
```

### csharp
```csharp
if (x > 0)
{
    Console.WriteLine(x);
}
```

### cpp
```cpp
#include <iostream>

int main()
{
    if (x > 0)
    {
        std::cout << x << std::endl;
    }

    return 0;
}
```

### fortran77
```fortran77
      PROGRAM EXAMPLE
  IF (X > 0) THEN
    PRINT *, X
  ENDIF
      END
```
